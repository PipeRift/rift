# Binary Operators

Binary operators operate two values to obtain a third one.

| Symbol | Name | Nodes | Code |
| --- | --- | --- | --- |
|  |  | 
| **Mathematic** |  |
| + | Add | | |
| - | Subtract |
| * | Multiply |
| / | Divide |
| % | Module |
|  |  |
| **Comparisons** |  |
| == | Equal |
| != | Not Equal |
| > | Greater |
| < | Less |
| >= ≥ | Greater or Equal |
| <= ≤ | Less or Equal |
|  |  |
| **Logical** |  |
| && | and |
| \|\| | or |
| & | bitwise and |
| \| | bitwise or |
| ^ | exclusive or (xor) |
